# 豆瓣图书推荐系统 python629

## 项目概述

豆瓣图书推荐系统是一款基于Python开发的应用，结合了Django框架和MySQL数据库，前端使用Vue.js技术。系统致力于为用户提供稳定的图书信息推荐服务。

## 技术栈

- 后端：Python Django
- 数据库：MySQL
- 前端：Vue.js

## 功能模块

- **用户模块**
  - 登录注册
  - 个人中心
- **图书信息模块**
  - 数据爬取
  - 小说信息管理
  - 图书借阅管理
  - 图书资讯管理
- **数据分析**
  - 可视化大屏
- **系统管理**
  - 用户管理
  - 系统简介

## 系统角色

- 普通用户：可浏览图书信息，进行注册、登录、借阅图书等操作
- 管理员：负责图书信息管理、用户管理及系统运行监控

## 运行环境

- 后端：Python 3.8+，Django 3.x，MySQL 8.x
- 前端：Vue.js 2.x，兼容现代浏览器

## 部署步骤

1. 环境准备：安装Python、MySQL、Node.js等
2. 数据库配置：创建数据库及用户，设置权限
3. 后端部署：通过Django的manage.py脚本进行数据库迁移及静态文件收集
4. 前端部署：使用Vue.js构建工具编译打包静态资源
5. 配置Web服务器：如Nginx，并指向后端服务及前端静态文件

## 目录结构

```
/douban_books_recommendation
|-- /backend              # 后端代码
|   |-- /apps              # Django应用
|   |-- /static            # 静态文件
|   |-- /templates         # 模板文件
|   |-- ...
|-- /frontend             # 前端代码
|   |-- /src               # Vue.js源代码
|   |-- /public            # 公共文件
|   |-- ...
|-- /data                  # 数据相关
|-- /docs                  # 文档
|-- ...
```
